# MERN Currency Converter

A full-stack currency converter built with React, Vite, Node.js, Express, MongoDB, and JWT authentication.

## Repository Setup

1. Clone the repository:
   ```bash
git clone https://github.com/<username>/<repository>.git
cd currency-converter
```
2. Install backend dependencies:
   ```bash
cd backend
npm install
```
3. Install frontend dependencies:
   ```bash
cd ../frontend
npm install
```

## Local Development

### Backend

```bash
cd backend
npm run dev
```

### Frontend

```bash
cd frontend
npm run dev
```

## Production Build

```bash
cd frontend
npm run build
```

## GitHub Repository Instructions

1. Create a new repository on GitHub.
2. Add the remote and push the initial commit:
   ```bash
git init
git add .
git commit -m "Initial commit: add MERN currency converter project"
git branch -M main
git remote add origin https://github.com/<username>/<repository>.git
git push -u origin main
```

## Git Commit Examples

- `git commit -m "Add frontend and backend structure for currency converter"`
- `git commit -m "Update API client to use environment variable for backend URL"`
- `git commit -m "Add GitHub Actions CI workflow and deployment docs"`
